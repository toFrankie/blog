## 基础

### 三种模式

- 普通模式：执行命令
- 插入模式：编辑文本
- 视图模式：高亮文本等


```
i    # 进入插入模式
v    # 进入视图模式
esc  # 从其他模式返回普通模式
```

### 光标移动

基础：

```
h    # 左
j    # 下
k    # 上
l    # 右
```
